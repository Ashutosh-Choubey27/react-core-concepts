
# What is React?

React is an **open-source JavaScript library** used to build **user interfaces**, especially **single-page applications (SPAs)**.

It was developed by **Facebook (Meta)** and focuses on building **reusable UI components**.

---

## Why React Exists?

Before React:
- Web pages were mostly **multi-page**
- Every user action caused a **full page reload**
- UI updates were slow and hard to manage

React solves this by:
- Updating only the **required part of the UI**
- Making applications **fast, scalable, and maintainable**

---

## Key Features of React

# 1. Component-Based Architecture
- UI is divided into **small reusable components**
- Each component manages its own logic and UI



# 2. Virtual DOM
 - React creates a Virtual DOM (a lightweight copy of the real DOM)

- It compares changes using a diffing algorithm

- Only the changed part is updated in the real DOM

✅ Result: Better performance

# 3. Declarative UI

- Instead of telling React how to update UI, we tell what the UI should look like.

- <p>{isLoggedIn ? "Welcome" : "Please Login"}</p>

- React handles the rest.


# 4. One-Way Data Flow

- Data flows from parent → child

- Makes debugging and state management easier
